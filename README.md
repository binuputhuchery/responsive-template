## Installation
`$ npm install`

### Requirements
* npm
* node
* gulp and its plugins

## Usage

```python
#Taskrunner

gulp # run all tasks 'styles, scripts, image, copy and watch'
gulp styles # Compiling sass
gulp images # Optimizing images
gulp scripts # Minifying JS
gulp copy # Copy markup and vendor files
```
